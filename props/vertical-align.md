#### Вертикальное выравнивание элемента внутри его контейнера. 
Применяется к родительскому контейнеру с потомками. 

Значение `vertical-align` может быть одним из следующих:

- `top`: По верхней границе контейнера.
- `middle`: По середине контейнера.
- `bottom`: По нижней границе контейнера.

- `text-top`: По верхней границе базовой линии текста.
- `baseline`: По базовой линии текста.
- `text-bottom`: По нижней границе базовой линии текста.

`vertical-align` также может принимать значения в процентах или фиксированных единицах измерения.

-------
Важно отметить, что `vertical-align` не всегда работает для всех типов элементов. 

-> Например, для блочных элементов это свойство может не иметь эффекта.
Используйте в блочных элементах `display: flex` и свойство `align-items` для выравнивания элементов внутри контейнера по вертикали.
```css
.container {
  display: flex;
  align-items: center; /* Выравнивание по центру */
}
```
